# Docker mcpe_viz

Simple container for [mcpe_viz](https://github.com/Plethora777/mcpe_viz)

## Getting started

### Build

```
bin/build_docker.sh
```

### Configure

Add world folders to `data/worlds/`

### Generate map

```
bin/docker_run.sh
```

### Run web server

```
bin/docker_run_http.sh
```
